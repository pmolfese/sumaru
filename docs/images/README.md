# README screenshots

Drop interface screenshots here with these exact names so the top of the
project `README.md` renders them. PNG preferred; aim for ~1200px wide.

| File | What to capture |
|---|---|
| `hero-surface-overlay.png` | The money shot: an inflated brain with a thresholded statistical overlay and the colorbar visible. Used as the hero banner. |

| `roi-drawing.png` | The ROI drawing controller with a couple of drawn ROIs on the surface. |

Until a file exists its image link in the README will show as broken on GitHub —
that's expected scaffolding. Capture with `r` (single view) in the viewer, or a
normal OS screenshot of the window.
